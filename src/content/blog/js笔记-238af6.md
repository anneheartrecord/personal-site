# js笔记

1.array

find:

  * 返回第一个满足条件的元素
  * 对每个元素都会执行
  * 如果没有满足条件的，则返回undefined
  * 空数组不会执行
  * 不会修改原数组

2.箭头函数

  * 本质是一种更简单方便的函数语法
  * `() => = function()`，当函数体只有一行的时候，`() => = function(){}`
  * 普通函数的this代表的是函数调用方，箭头函数的this代表的是函数的拥有者（定义时就被确定）

3.拼接字符串的几种方式

  * +号
  * ``${var}``模版字符串
  * `x.concat('', y)`
  * `array.join`拼接数组字符串

4.Promise

  * then 成功
  * catch 失败
  * finally 都会做
