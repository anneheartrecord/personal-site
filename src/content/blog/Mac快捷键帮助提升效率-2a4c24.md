# Mac快捷键帮助提升效率

## 快捷键

切换输入法

短按中/英键为切换输入法，长按则为切换大小写，同时control+ space 也是切换输入法。

苹果键盘中有一个command键，比较特殊，相当于windows的win键，下面有一些常用的command快捷键大全

  * command + z 撤销
  * command + c 拷贝
  * command + v 粘贴
  * command + a 全选
  * command + x 文字剪切
  * command + f 查找
  * command + w 关闭当前页面
  * command + y 访问历史记录
  * command + m 最小化当前页面
  * command + h 隐藏当前页面（右下角程序看不见）
  * command + q 关闭所有页面
  * command + n 新建窗口
  * command + t 新建页面
  * option + command + h 隐藏其他页面
  * command + r 刷新页面
  * command + l 地址栏
  * shift + command + 3 全屏截图
  * shift + command + 4 选择区域截图
  * control + command + shift + 3 全屏截图复制到粘贴板
  * control + command + shift + 4 选择区域截图复制到粘贴板
  * command + delete 删除到回收站
  * option + command + delete 彻底删除回收站
  * fn + f 退出全屏

## 触控板快捷方式

一个手指点击，选中

一个手指点按，相当于鼠标左键

两个手指点按，相当于鼠标右键

两指上下左右滑动，对应方向推进页面

两指分开旋转，可以旋转图片，缩放页面大小，效果等同于command + 和 command -

三指左右滑动，可以在全屏应用中切换

三指向上滑动，打开正在使用的所有软件窗口

四指捏合，调出应用程序窗口

四指张开，显示桌面
