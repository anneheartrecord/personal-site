很多人对做自媒体有极大的误解，认为这只是为了赚快钱，或者多一份收入。

但如果你只盯着眼前的流量收益，大概率坚持不到一个月就会放弃。

我认为持续输出的价值有以下七点，其中金钱收益反而是最微不足道的。

## 1️⃣ 输出是最高效的学习

这是输出最底层、却被大多数人忽略的价值，也是费曼学习法的终极实践。

心理学上有个词叫做能力错觉，这也是为什么很多人沉迷于看很多书、买很多课。

因为买这个动作能带来虚假的成长感。

你以为学完了等于懂了，但其实你不懂，得到了某些信息只能说明教你的老师是真的懂了。

只有彻底内化了文章和课程的内容，并且这些内容真正改变了你的生活，成为了你的一部分，才算是真的懂了。

对于新学的知识：采取 Build in public 的模式，把你的踩坑日常、学习笔记稍微润色一下然后发出来。写推文的过程，会强迫你把模糊的直觉转化为精确的文字。只要你写不出来，就说明你根本没懂。

对于旧知识：你的大脑里塞满了碎片化的思绪。为了写出一篇合格的推文，你必须强迫自己梳理一遍旧知识，每一次结构化的输出，都是对原有知识的再一次深度重构。

所以不要怕说错话，不要怕自己发的东西很 low。

高质量学习才是输出的第一目的。每一次输出都是一次高质量的学习，当你真正内化了这一点，你反而不会太在意数据和流量。

## 2️⃣ 培养 Sale 能力

Naval 说过，赚钱需要两种能力：Build 和 Sale。

不论你是程序员、设计师还是运营，只要你还在打工，你其实就只具备了 Build 的能力。因为能力的缺失，你也只能通过加入到某个公司工作来间接地为社会提供价值。

但当你在社交媒体上积累了一定的关注度，你就掌握了媒体杠杆。

你的每一个想法、每一行代码、每一个产品，都会被几千人、几万人甚至几十万人看到。

只要一个人兼具 Build 和 Sale 这两项能力，上班对你来说就只是一个可选项了。因为你形成了商业逻辑的闭环，你成为了一家最小粒度的一人公司。

## 3️⃣ 培养 Storytelling 能力

在你做账号的过程中，你肯定还是得花一部分精力和算法、内容传播逻辑打交道。

算法的背后，就是极其精准的人性。

你需要构思如何写出一个抓人眼球的标题和开头，如何调动情绪，如何排版才能让读者不划走。这些动作其实都在训练自己的 Storytelling 能力。

这个能力是跨界且降维的，当你把这种能力带回主业，你会发现：

1. 你能写出万赞的爆款推文，你就能写出让老板惊艳的汇报 PPT。
2. 你能勾住陌生网友的注意力，你就能在跨部门谈判、客户提案和晋升答辩中游刃有余。

因为你已经掌握了如何讲出一个美妙且吸引人的故事，而人类天生就爱听故事。

## 4️⃣ 数字资产

有个非常残酷的现实：无论你在公司干了多少年、加过多少班、做过多大的项目，离职的那一天，你什么都带不走。

你只能带走一张薄薄的简历。

这张简历能承载的东西太少了，而你在 X 上的账号刚好能作为补充。

它是完完全全属于你的作品，它向所有人展示了你是谁、如何思考的。

每一篇高质量的推文都是在为你积蓄势能，每一篇推文都能吸引来一部分志同道合的同伴。

## 5️⃣ 帮助链接，拓宽信息源

这是我在 X 输出之后才感受到的巨大收益。

之前我的社交圈子也几乎全是和我相同路径的大厂程序员，彼此之间能够交换的信息非常有限。

但在 X 上，你输出什么样的内容，算法就会自动帮你匹配什么样的人。

同时你的内容又降低了链接的门槛。你的思考深度、认知层次会通过你的内容完全展示出来。

## 6️⃣ 建立多维评价体系

如果你只有一份工作，你的自我认同感是极度脆弱的。

你很容易被 KPI、晋升和领导的脸色困住，陷入自我怀疑。

但当你拥有了自媒体，你会获得多维度的收入和认可。你会更客观地看待自己，真正聚焦于自己的成长，而不是外界赋予你的虚名和头衔。

当你发现你的价值是由市场决定的，而不是某个上司、公司赋予的虚名。这种掌控感和满足感，是任何高薪工作都给不了的。

## 7️⃣ 告别单点收入

最后，才是最直接的收入问题。

在计算机系统架构里，有一个词叫单点故障，只要这一个节点坏了，整个系统就会崩溃。

依靠单一的死工资，就是普通人最大的单点收入风险。

一旦遭遇裁员、降薪、生病，你的现金流可能会瞬间归零。

通过 X 的流量分成、广告赞助，甚至是你自己开发的付费产品，你为自己接入了一条全新的收入管道。

这条管道在初期可能只能给你赚杯咖啡钱，但它完全遵循复利效应，能够慢慢越滚越大。

最后我想说，输出，是一场稳赚不赔的自我投资。

哪怕最后没能赚到大钱，你在这个过程中被动升级的结构化思维、搞流量能力、数字资产和高质量人脉，也足以让你完成一次进化。
